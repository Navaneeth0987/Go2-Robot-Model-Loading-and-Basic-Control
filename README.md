# Go2 Quadruped Robot – Model Loading (MuJoCo)

## 📌 Overview
This project demonstrates loading and initializing the Unitree Go2 quadruped robot model using a simulation environment.

It focuses on setting up the robot model and establishing a basic control interface using Python.

---

## 🧠 Project Structure

Go2-Robot-Model-Loading/
│
├── src/        # Main simulation and model loading code  
├── sdk/        # SDK interface  
├── config/     # Configuration parameters  
├── README.md  

---

## ⚙️ Features
- Load Go2 robot model in simulation  
- Basic control interface setup  
- SDK integration  
- Configuration-based initialization  

---

## 🛠️ Technologies Used
- Python  
- MuJoCo  
- Unitree SDK  

---

## 🚀 Purpose
This project serves as a foundational step towards developing advanced locomotion and control algorithms for quadruped robots.
